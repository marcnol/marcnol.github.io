Alarm

Indoor alarm activation : 6:30am-21pm

A beep sounds 15min before the alarm is triggered.

Deactivate alarm that is going to be activated

  1. At 20h45 it starts beeping

  2. Type the code once (ask code to manager), then wait, alarm says it is armed

  3. Then type it once more, the alarm should say it is disarmed

  4. Repeat the procedure for another zone

  5. When you leave, just type the code once to activate it.

We should call SCUTUM (02 number), though if you don’t they don’t seem to say anything.

Disable WE Alarm:

  1. Sign up on the google sheet

  2. Wait until the alarm is activated (21:15 => [4] : Activated).

  3. Code to deactivate zone 4 (4: Disabled)

  4. Inform the SCUTUM monitoring company (: 08 25 34 20 00 or 02 35 07 16 16) indicating :

    • Name and surname
    • Coordinates of the building : DR 08
    • Password : (Ask your manager)
    • Approximate duration of your presence in the building

  5. Fill in the I/O log on the counter at the RD reception desk.

  6. Initially reactivate the alarm with the code assigned for the R+1 of the DR ([4]: Enabled)

Note: To deactivate the alarm in the evening, instead of waiting until 9:15pm, Baptiste will activate and deactivate the alarm as soon as he hears the beep at 8:45pm. He does not call the company…
