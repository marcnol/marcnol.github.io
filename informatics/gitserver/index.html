Install git server

On the server side

Install git:

sudo apt-get install git-core
sudo useradd git
passwd git

and enter the desired password.

To get SSH keys, you run this in the local machine

ssh-keygen -t rsa
cat ~/.ssh/id_rsa.pub | ssh git@grey "mkdir -p ~/.ssh && cat >>  ~/.ssh/authorized_keys"

Need to run to create repository

mkdir /home/git/Repositories/newrepo
git init --bare

On the local machine

The first time you run this:

mkdir /home/git/Repositories/newrepo

git init 

git add .

git commit -m "message" -a

git remote add origin ssh://git@grey/home/marcnol/Repositories/newrepo.repo.git

git push origin master

For any subsequent update you push by doing:

git add .

git commit -m "message" -a

git push origin master

Cloning

git clone marcnol@grey/home/marcnol/Repositories/newrepo.repo.git
